 # 🛠️ Book Bound

"A responsive MERN stack web application built with React and NextJs, featuring a clean UI powered by Tailwind CSS and DaisyUI. Designed for high performance and seamless user experience."

## 📖 Project Overview
"A subscription-driven digital library application that offers registered users unlimited access to a wide range of books. It features a seamless monthly membership model, ensuring a cost-effective and user-friendly reading environment for everyone."

 ## 🚀 Live Demo
Check out the live project here:  
[Keen Keeper Live Link](https://book-bound-next-js.vercel.app/)

## ✨ Key Features
* **User Authentication:** Secure login and registration system for readers.
* **Subscription Model:** Monthly membership plans to unlock premium book collections.
* **Extensive Book Library:** Categorized books (e.g., Story, Science, Tech) with a powerful search and filter system.
* **Admin Dashboard:** manage user subscriptions.
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop views.

## 💻 Tech Stack
This project was built using the following technologies:
* **HTML5 & CSS3** - For structural semantic layout.
* **Tailwind CSS** - For modern, utility-first styling.
* **Daisy UI** - For use useble Component.
* **JavaScript (ES6+)** - For dynamic functionality and interactivity.
* **NextJS** - For SEO Friendly Dynamic Route Base Site.
* **React -Toastify** (NPM Package) - For Use Beutifull Toast In Button Clicking.


## ⚙️ npm Packages
* **Security with Better-Auth:** Implemented robust and developer-friendly authentication to ensure user data privacy and seamless session handling.
* **User Feedback with React Toastify:** Integrated real-time notifications to provide instant feedback on user actions like login, subscription updates, and alerts.
* **Visual Appeal with React Fast Marquee:** Utilized smooth, high-performance scrolling animations to highlight featured authors and trending books on the homepage.

## ⚙️ Local Setup Guide

Follow these steps to set up the project on your local machine:

### 1. Prerequisites
Make sure you have **Node.js** and **npm** installed on your system. You can check this by running:

```bash
node -v
npm -v
```

### 2. Clone the Repository
First, clone the project from GitHub to your local computer:

```bash
git clone https://github.com/mdmukhlesurrahman1/BookBound-NextJS.git

```
### 3. Navigate to the Directory
Move into the project folder:

```bash
cd b13-assignment-7

```
### 4. Install Dependencies
Install all the necessary packages required for the project:

```bash
npm install
```

### 5. Run the Project
Start the development server:

```bash
npm run dev
```
